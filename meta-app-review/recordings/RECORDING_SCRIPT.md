# Screen Recording Script
## What to Record for Meta App Review

### Recording Setup
- Use a screen recorder (OBS, Loom, or phone recording)
- Record in 1080p if possible
- Keep it 30-60 seconds max
- Show the Facebook page name clearly
- Show real messages, not test data

---

### SCENE 1: Customer Sends Message (10 seconds)
1. Open Facebook Messenger
2. Go to your test page "Sahla4Eco Test"
3. Send a message: "Hi, do you have this product in stock?"
4. Show the message being sent

### SCENE 2: Bot Receives and Responds (15 seconds)
1. Switch to your dashboard (show webhook receiving the message)
2. Show the message appearing in your system
3. Show the AI generating a response
4. Switch back to Messenger
5. Show the bot's reply appearing

### SCENE 3: Order Confirmation (15 seconds)
1. In your dashboard, show an order being created
2. Show the order confirmation being sent via Messenger
3. Switch to Messenger
4. Show the order confirmation message received by customer

### SCENE 4: Delivery Update (10 seconds)
1. In your dashboard, change order status to "Shipped"
2. Show the delivery update being sent
3. Switch to Messenger
4. Show the shipping notification received

---

### Recording Checklist
- [ ] Facebook page name visible
- [ ] Real customer message (not "test123")
- [ ] Bot response appears within 5 seconds
- [ ] Order confirmation shows real details
- [ ] Delivery update shows tracking info
- [ ] No sensitive data visible (passwords, tokens)
- [ ] Total duration: 30-60 seconds

---

### Common Mistakes to Avoid
❌ Recording too long (over 2 minutes)
❌ Showing test/placeholder messages
❌ Not showing the Facebook page name
❌ Recording in low quality
❌ Showing error messages or bugs
❌ Including sensitive data in recording

### What Meta Wants to See
✅ Real customer interaction flow
✅ Bot responds automatically
✅ Messages are helpful and relevant
✅ No spam or promotional messages
✅ Order updates are accurate
✅ Customer privacy is protected
